## EDA and Data Wrangling

This folder contains the notebooks used to explore, clean, and prepare the Falcon 9 launch dataset for modeling. The workflow includes initial data inspection, creation of the binary classification target, SQL‑based exploration, and visual analysis.

</br>

## 1. Data Wrangling (`DataWrangling.ipynb`)

- Perform initial exploratory checks on dataset structure and quality  
- Clean and standardize key variables  
- Create the binary outcome feature used for classification  
- Prepare the dataset for deeper analysis and feature engineering


</br>

## 2. EDA with SQL (`EDAwithSQL.ipynb`)

- Use SQL queries to explore the dataset  
- Explore targeted questions about launch records  
- Validate assumptions and uncover patterns not immediately visible through Pandas alone  
- Produce interpretable summaries that guide later modeling decisions



</br>

## 3. Data Visualization (`Visualization.ipynb`)

- Visualize relationships between key variables  
- Identify potential predictors for landing success  
- Support feature engineering decisions through plots and charts



</br>

## Notebook Index

- **DataWrangling.ipynb** — EDA and creation of the binary outcome feature  
- **EDAwithSQL.ipynb** — SQL‑based exploration of the dataset  
- **Visualization.ipynb** — Visual analysis and feature engineering insights
